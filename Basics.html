Getting started with React

BASE CLASS 'COMPONENT':

React consists of a base class called Component

In the beginning of the code, you have to export it

It contains of elements which can be exported and used in the code




APP CLASS:

The App class is a component of the baseClass "Component"

We need to export it at the beginning of the code

We have to draw the existing components like Button, View, Text from the App class

The App Class can contain only one function at a time

For example, render();




RENDER FUNCTION:

The render function is a function which can contain only one function inside it

example: return function




RETURN FUNCTION:

The return function displays whatever we enter inside it as the final output

It can contain only one component inside it

example: It can hold only button, or text, or whatever elemt you choose




VIEW component:

The view component can hold more than one component inside it

This can be used if you want to display more than one component in your content

It has a opening and closing tag




BUTTON COMPONENT:

The button component is very similiar to the one in html

It is used to create Buttons

We can name it by using its property "title"

Refer Tutorials for examples

We can customize its color using the "color" property

It has a opening and closing tag




TEXT COMPONENT:

The text component displays text

It is very similiar to the one in html

It has a opening and closing tag
